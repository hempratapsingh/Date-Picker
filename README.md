
📅 Datepicker UI (HTML & CSS)

This project is a **static Datepicker User Interface** built using only **HTML and CSS**.
It is designed to look like a calendar component and serves as a foundation for building a fully functional datepicker in the future.

🚀 Features

- 📆 Calendar-style layout (7-column grid)
- 🎯 Month and navigation arrows (UI only)
- 🗓️ Days of the week header
- 🔢 Date grid (1–31)
- 🧩 Clean grid borders like a real calendar
- ✨ Hover effects on dates
- 🎨 Simple and modern design

🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)

📂 Project Structure

```
datepicker-ui/
│── index.html
│── style.css
│── README.md
```

📸 Preview

A simple calendar UI with:

- Month name at the top
- Days (Sun–Sat)
- Dates arranged in grid format

🎯 Purpose of the Project

This project helps in practicing:

- CSS Grid layout
- Element positioning
- UI design fundamentals
- Styling reusable components

It is **not functional yet**, but can be upgraded using JavaScript.

🔮 Future Improvements

- Add JavaScript for:
  - Month switching
  - Dynamic dates
  - Date selection

- Highlight current date automatically
- Add animations and transitions
- Make it fully responsive

💡 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize styles in `style.css`

Project URL: https://roadmap.sh/projects/datepicker-ui

📌 Note

This is a **UI-only project**, so the buttons and dates are not interactive yet.
